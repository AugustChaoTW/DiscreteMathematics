# 離散數學 (Discrete Mathematics)

這是一個用 GitHub Pages 建立的離散數學學習資源網站。

## 🎯 專案內容

本網站包含以下主要主題：

- **邏輯與命題** - 命題邏輯、布林代數和推理規則
- **集合論** - 集合、關係、函數和運算
- **圖論** - 圖的結構、路徑、樹和著色
- **組合數學** - 計數原理、排列和組合
- **遞推關係** - 遞推方程和複雜度分析
- **形式語言與自動機** - 正則表達式和自動機

## 📂 檔案結構

```
DiscreteMathematics/
├── index.html           # 主頁面
├── style.css            # 樣式表
```

## 🚀 本地測試

如果你想在本地測試此網站，可以使用以下步驟：

### 使用 Python 內建伺服器

```bash
cd DiscreteMathematics
python3 -m http.server 8000
```

然後在瀏覽器中訪問 `http://localhost:8000`

### 使用 Jekyll（需要先安裝）

```bash
bundle install
bundle exec jekyll serve
```

## 📝 如何修改內容

1. 編輯 `index.html` 修改頁面結構
2. 編輯 `style.css` 修改樣式
3. 編輯 `script.js` 修改互動功能
4. 修改後 commit 並 push 到 GitHub，網站會自動更新

## 🔧 自訂設定

在 `_config.yml` 中可以修改：

- `title` - 網站標題
- `description` - 網站描述
- `theme` - Jekyll 主題
