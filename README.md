# 離散數學教學遊戲 (Discrete Mathematics Teaching Game)

這是一個以離散數學為主的數學教學遊戲專案，透過互動式問答幫助學生學習離散數學的核心概念。

## 功能特色

本遊戲涵蓋四大離散數學主題：

### 1. 集合論 (Set Theory)
- 集合的基本運算：聯集、交集、差集
- 冪集合的概念
- 子集與空集合的性質

### 2. 邏輯 (Logic)
- 命題邏輯的基本運算符
- 真值表與邏輯等價
- 德摩根定律
- 推理規則

### 3. 圖論 (Graph Theory)
- 圖的基本概念
- 歐拉路徑與歐拉回路
- 樹的性質
- 完全圖

### 4. 組合學 (Combinatorics)
- 排列與組合
- 計數原理
- 組合數的性質

## 如何使用

1. 在瀏覽器中開啟 `index.html` 檔案
2. 從主選單選擇想要學習的主題
3. 閱讀每個主題的說明
4. 回答問題並獲得即時回饋
5. 累積分數，追蹤學習進度

## 技術架構

- **HTML5**: 網頁結構
- **CSS3**: 美觀的使用者介面與響應式設計
- **JavaScript**: 互動邏輯與遊戲機制
- **LocalStorage**: 儲存學習進度與總分
